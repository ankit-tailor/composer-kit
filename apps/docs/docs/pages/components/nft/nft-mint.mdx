# NFTMint

The `NFTMint` component is a part of the Composer Kit UI library for NFTs. It renders a minting interface that lets users mint an NFT directly from your application. When placed within an `NFTCard`, it integrates seamlessly with other NFT components such as metadata, image, and token ID displays. The component derives its context from the parent `NFT` component, which supplies the required contract address and token ID.
